Motivation

This web page is created to provide a visual representation of the data and the relationships between the features in the dataset. The dataset used is the Student Performance Data Set from the UCI Machine Learning Repository. The dataset contains student achievement data from two Portuguese schools. The data includes student grades, demographic, social and school related features.
